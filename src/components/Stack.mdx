# Stack

## How to install

```bash
  npm install @bedrock-layout/stack
```

## Properties

| Property | Type                                    | Default |
| -------- | --------------------------------------- | ------- |
| gutter   | One of: `none, xs, sm, md, lg, xl, xxl` | `md`    |
