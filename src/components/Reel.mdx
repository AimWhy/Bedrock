# Reel

## Use case

Scrolling is a popular and natural way to interact with web content. The Reel is
designed to organize content into scrollable flat lists with convenient scroll breakpoints.
Optimally used with the `tabIndex` attribute.

## How to install

If you haven't already, install `styled-components`:

```bash
  npm install styled-components
```

Then install the stack component

```bash
  npm install @bedrock-layout/reel
```
