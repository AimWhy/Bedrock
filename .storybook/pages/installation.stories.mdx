# How to install

Bedrock Layouts is built on top of [styled-components](https://styled-components.com). If you would like to learn more about how to use styled-components, please refer to the [styled-components documentation](https://styled-components.com/docs)

Before you install any of the Bedrock Layout Primitives, you will need to install `styled-components` first:

```bash
npm install styled-components

# or

yarn add styled-components
```

Then can install any of the layout primitives you need; for example, to install the `Stack` component, you would run:

```bash
npm install @bedrock-layout/stack

# or

yarn add @bedrock-layout/stack
```

Finally, you import the component into your app like this:

```javascript
import { Stack } from '@bedrock-layout/stack';
```
