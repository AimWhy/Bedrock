## Getting Started

Lets start by creating our boilerplate React application

```bash
yarn create react-app bedrock-layouts
cd bedrock-layouts
```

Bedrock Layouts is built on top of [styled-components](https://styled-components.com). If you would like to learn more about how to use styled-components, please refer to the [styled-components documentation](https://styled-components.com/docs)

Before you install any of the Bedrock Layout Primitives, you will need to install `styled-components` first. Let's start out by installing our the `Stack` component:

```bash
yarn add styled-components @bedrock-layout/stack
```

## App.js

Replace the code in App.js with the following example:

```jsx
import { Stack } from "@bedrock-layout/stack";

function App() {
  return (
    <Stack gutter="md">
      <header>
        <h1>Coco Le Belle</h1>
      </header>
    </Stack>
  );
}

export default App;
```

Finally, let's start the development server by running `yarn start`. Navigate to `http://localhost:3000/` in your browser and you should see the following output:

<Center centerChildren>
