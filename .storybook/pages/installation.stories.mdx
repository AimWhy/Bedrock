# How to install

If you haven't already, install `styled-components`:

```bash
npm install styled-components

# or

yarn add styled-components
```

Then install the component; for example, to install the Stack:

```bash
npm install @bedrock-layout/stack

# or

yarn add @bedrock-layout/stack
```

Finally, you import the component.

```javascript
import { Stack } from '@bedrock-layout/stack';
```
