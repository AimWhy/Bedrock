## Zero Config

Bedrock Layout Primitives are built using [styled-components](https://styled-components.com/). This allows Bedrock Layout Primitives to be used in any app without any extra setup. Just `yarn add` the desired package(s) and start using them.

</div>
<div>

## Composable Layouts

Bedrock Layout Primitives are built to allow you to compose them together to create your own custom layouts. Many "complex" layouts can be created by composing Bedrock Layout Primitives. Learn more about layout composition in the [Introduction](/docs/getting-started-introduction--page#what-is-composition) section.

</div>
<div>

## Use With Any Design System

You can use Bedrock Layout Primitives with any design system. Bedrock Layout's' spacing scheme can be easily overridden to match your design system. [Check out the instructions on how to integrate the Bedrock Layout Primitives into your Design System.](/docs/getting-started-spacing--page#integrating-with-your-design-system)

</div>
<div>

## CSS Only Version

Don't want to use React or styled-components? Bedrock Layout Primitives can be used in any app with just CSS. [Check out the instructions on how to use Bedrock Layout Primitives in your app.](/docs/css-only-a-css-only-version--page)

</div>
