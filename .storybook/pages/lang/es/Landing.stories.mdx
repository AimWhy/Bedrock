## Sistema de diseño de componentes

<Split fraction='1/3' gutter='xxl' switchAt="42rem" >
    <Reel style={{flexDirection:'column'}}>

```jsx
export function Hero() {
  return (
    <Stack>
      <Inline>
        {/* */}
      </Inline>
      <Split>
        <Cover>
          <Stack>
            <h1>{/* */}</h1>
            <p>{/* */}</p>
            <Inline>
              <button>
                {/* */}
              </button>
              <button>
                {/* */}
              </button>
            </Inline>
          </Stack>
        </Cover>
        <Frame>
          <img />
        </Frame>
      </Split>
    </Stack>
  );
}

```

  </Reel>
  <section>

La mayoría de los diseños que usamos en internet no son tan únicos. Frecuentemente pueden ser divididos en una pequeña cantidad de patrones de diseño. Es la combinación de estos patrones los que hacen el diseño gráfico general.

Los diseños de Bedrock Layout Primitives han sido construidos con esta idea en mente. Cada componente está construido para ejecutar bien un patrón de diseño. Con la combinación de estos componentes individuales puedes crear tus propios diseños. Muchos de los diseños ¨complejos¨ que hacemos en internet pueden ser creados combinando una cantidad de Bedrock Layout Primitives. Hay [más ejemplos de recetas de diseño en nuestra sección de ejemplos.](/docs/examples-web-dev--12-span-grid#layout-patterns)

Puedes aprender más sobre la composición de diseños en la [Página de introducción](/docs/getting-started-introduction--page).

## Componentes faltantes en tu sistema de diseño

<Split gutter='xl' switchAt="40rem">
  <section>

Tu diseño estándar tiene muchos componentes, pero sólo unos pocos de ellos están dedicados a colocar esos componentes en tu página. Bedrock Layout Primitives está diseñado para ayudarte a ampliar tu diseño, brindando un conjunto de primitivas de diseño que pueden ser combinadas para crear un diseño completo.

Puedes usar Bedrock Layout Primitives con cualquier diseño. El esquema de espaciado de Bedrock Layout puede ser fácilmente sobrescrito para que coincida con tu diseño. [Revisa las instrucciones para integrar los Bedrock Layout Primitives en tu diseño.](/docs/getting-started-spacing--page#integrating-with-your-design-system)

## No solamente para React

<Split fraction='2/3' gutter='xl' switchAt="43rem">
    <Reel style={{flexDirection:'column'}}>

```jsx
<header data-bedrock-cover>
  <nav data-bedrock-inline-cluster>
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
  </nav>
  <div
    data-bedrock-cover-centered
    data-bedrock-stack
  >
    <h1>Coco Le Belle</h1>
    <div
      data-bedrock-split='fraction:1/3'
    >
      <p>
        <strong>Coco Le Belle</strong>
        is an online fashion boutique.
      </p>
      <img src='logo.png' />
    </div>
  </div>
</header>

```

  </Reel>
  <section>

¿No quieres usar React? Tenemos lo que necesitas. Bedrock Layout Primitives es también un Framework CSS. Construido usando atributos de datos, sigue siendo un componente como experiencia de desarrollador sin que afecte a tus clases de nombre. Cualquier aplicación que utilice HTML y CSS puede aprovechar Bedrock Layouts. [Revisa las instrucciones para usar Bedrock Layout Primitives en tu aplicación.](/docs/css-only-a-css-only-version--page)

Una vesion de solid.js está en marcha. Puedes revisar Solid-Bedrock en [solid.bedrock-layout.dev](https://solid.bedrock-layout.dev)

  </section>
