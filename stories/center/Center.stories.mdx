# Center

## Use case

The `Center` component is designed to center and clamp its width at a predefined value. By default, this value is set to the `medium` breakpoint. You can also center the children and text alignment as well.

Some examples of where you might use a `Center` component is:

- [Header Sections](/docs/examples-basic--docs#header-sections)

## Installation

```bash
yarn add @bedrock-layout/center
```

## API

<ArgsTable />

## maxWidth

The Center component will clamp the width at a defined `maxWidth` and center itself in its context. Please note that the `maxWidth` prop represents the `max-width` of the children and not the Center component itself. In the example shown below, the `maxWidth` is set to `75%` of the parent component's width.

## centerChildren

You can also center the children by adding a `centerChildren` prop. Also, the max width of the children is set to 75%.

<Canvas withSource="open">
  <Story name="centerChildren">

      <div style={{ width: "75%" }}>

## data-attribute

For styling purposes, you can select `data-bedrock-center`.
