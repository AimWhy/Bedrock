# reel.css

This is a CSS Only version of the [Reel package](/docs/spacer-components-reel--docs).

## Use case

Scrolling is a popular and natural way to interact with web content. The Reel is
designed to organize content into scrollable flat lists with convenient scroll breakpoints.

## Usage

First you need to import the appropriate stylesheet. The reel is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import "@bedrock-layout/css/lib/bedrock-layout.min.css";

// or

import "@bedrock-layout/css/lib/components/reel.min.css";
```

Then you add the `data-bedrock-reel` attribute to the desired element.

## gutter

The `gutter` prop defines the gutter size between elements. Bedrock has adopted the [open props sizes as a default spacing scheme](https://open-props.style/#sizes).

You can adjust the space between the child elements by setting the `gutter:` value to one of the open props size values, such as `gutter:size3`.
You can also set the gutter value by passing the `--gutter` property in the inline style directly.

```jsx
<div data-bedrock-reel="gutter:size3">
  <Component />
  <Component />
</div>

<div data-bedrock-reel style={{ "--gutter": "var(--space-2x)" }}>
  <Component />
  <Component />
</div>

<div data-bedrock-reel style={{ "--gutter": "2rem" }}>
  <Component />
  <Component />
</div>
```

## snapType

The `snapType` property sets the scroll snap type. The default snap type is `none`,
but you can set it to `proximity` or `mandatory` by setting the `snapType:` value
on the `data-bedrock-reel` attribute. For example, if you want to set the snap type
to be `mandatory` you would set the `data-bedrock-reel` attribute to `snapType:mandatory`.

```jsx
<div data-bedrock-reel="snapType:none">
  <Component />
  <Component />
</div>

<div data-bedrock-reel="snapType:mandatory">
  <Component />
  <Component />
</div>

<div data-bedrock-reel="snapType:proximity">
  <Component />
  <Component />
</div>
```
