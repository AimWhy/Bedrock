# stack.css

This is a CSS Only version of the [Stack package](/docs/spacer-components-stack--docs).

## Use case

The Stack is designed to literally 'stack' items on top of each other while maintaining a consistent gutter between each item.

## Usage

First you need to import the appropriate stylesheet. The stack is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import '@bedrock-layout/css/lib/bedrock-layout.min.css';

// or

import "@bedrock-layout/css/lib/components/stack.min.css";
```

Then you add the `data-bedrock-stack` attribute to the desired element.

## gutter

The `gutter` prop defines the gutter size between elements. Bedrock has adopted the [open props sizes as a default spacing scheme](https://open-props.style/#sizes).

You can adjust the space between the child elements by setting the `gutter:` value to one of the open props size values, such as `size3`. You can also set the gutter value by passing the `--gutter` property in the inline style directly.
