# split.css

This is a CSS Only version of the [Split package](/docs/spacer-components-split--docs).

## Use case

The Split component is designed to create a split layout based on a fractional ratio. The Split will enforce a standard spacing scheme through the gutter prop.

## Usage

First you need to import the appropriate stylesheet. The split is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import "@bedrock-layout/css/lib/bedrock-layout.min.css";

// or

import "@bedrock-layout/css/lib/components/split.min.css";
```

Then you add the `data-bedrock-split` attribute to the desired element.

## gutter

The `gutter` prop defines the gutter size between elements. Bedrock has adopted the [open props sizes as a default spacing scheme](https://open-props.style/#sizes).

You can adjust the space between the child elements by setting the `gutter:` value to one of the open props size values, such as `gutter:size3`.
You can also set the gutter value by passing the `--gutter` property in the inline style directly.

```jsx
<div data-bedrock-split="gutter:size3">
  <Component />
  <Component />
</div>

<div data-bedrock-split style={{ "--gutter": "var(--space-2x)" }}>
  <Component />
  <Component />
</div>

<div data-bedrock-split style={{ "--gutter": "2rem" }}>
  <Component />
  <Component />
</div>
```

## fraction

The `fraction` prop defines the fraction of the container width to use for the split.

By default the split will be a `1/2` split. You can change this by setting the `fraction:` value on the `data-bedrock-split` attribute.  
For example to set a `3/4` split ratio you would use `fraction:3/4`.

```jsx
<div data-bedrock-split="fraction:1/4">
  <Component />
  <Component />
</div>

<div data-bedrock-split="fraction:1/3">
  <Component />
  <Component />
</div>

<div data-bedrock-split="fraction:1/2">
  <Component />
  <Component />
</div>

<div data-bedrock-split="fraction:2/3">
  <Component />
  <Component />
</div>

<div data-bedrock-split="fraction:3/4">
  <Component />
  <Component />
</div>

<div data-bedrock-split="fraction:auto-start">
  <Component />
  <Component />
</div>

<div data-bedrock-split="fraction:auto-end">
  <Component />
  <Component />
</div>
```

## --minItemWidth

The `--minItemWidth` prop defines the minimum inline size of each child.

If the minimum inline size can not be maintained, it will move to a stacking layout.

```jsx

<div
  data-bedrock-split="fraction:2/3 gutter:size2"
  style={{ "--minItemWidth": "40ch" }}
>
  <Component />
  <Component />
</div>

<div
  data-bedrock-split="fraction:auto-start gutter:size2"
  style={{ "--minItemWidth": "30ch" }}
>
  <Component />
  <Component />
</div>
```

## --switchAt

If you set the `--switchAt` property, the inline will switch to a stacking layout if the width of the wrapper
element is below the threshold set by the `--switchAt` property.

This can be used in conjuction with `--minItemWidth` and will switch to a stacking layout if the either the
children cannot maintain the minItemWidth or the wrapper component's width falls below the `--switchAt` size.

```jsx
<div
  data-bedrock-split="fraction:2/3 gutter:size2"
  style={{ "--switchAt": "45rem" }}
>
  <Component />
  <Component />
</div>

<div
  data-bedrock-split="fraction:auto-start gutter:size2"
  style={{
    "--minItemWidth": "20rem",
    "--switchAt": "45rem",
  }}
>
  <Component />
  <Component />
</div>
```
