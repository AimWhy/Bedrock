# Grid

## Use case

The Grid component is designed to create a grid of items that will automatically wrap based on the `minItemWidth`

## How to install

If you haven't already, install `styled-components`:

```bash
  npm install styled-components
```

Then install the Grid component

```bash
  npm install @bedrock-layout/grid
```
