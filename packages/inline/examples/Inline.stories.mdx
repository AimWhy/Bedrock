# Inline

## Use case

The Inline component is designed to create consistent spacing around inline elements of variable width. Unlike the InlineCluster component,
the items in the Inline component will not wrap. The Inline component also allows you to specify a child that can stretch, filling the excess space.

## InlineCluster Props

The Inline component has all the same props as the InlineCluster component. (check the [InlineCluster component for more details](/?path=/docs/components-inlinecluster--api))

Below is an example of the Inline component with the `gutter` set to `xl` and both `justify` and `align` props set to `center`.

<Preview withToolbar>
  <Story name="InlineCluster Props">
    <Inline gutter="xl" justify="center" align="center">
      <Box style={{ height: 200 }} />
      <Box />
      <Box />
      <Box />
    </Inline>
  </Story>
</Preview>

## With stretch start

The `stretch` prop can be used to specify a child that will stretch to fill the remaining space.

<Preview withToolbar>
  <Story name="With stretch start">
    <h3>start</h3>
    <Inline gutter="lg" stretch="start">
      <Box />
      <Box />
      <Box />
      <Box />
    </Inline>
    <h3>end</h3>
    <Inline gutter="lg" stretch="end">
      <Box />
      <Box />
      <Box />
      <Box />
    </Inline>
    <h3>all</h3>
    <Inline gutter="lg" stretch="all">
      <Box />
      <Box />
      <Box />
      <Box />
    </Inline>
    <h3>2 index</h3>
    <Inline gutter="lg" stretch={2}>
      <Box />
      <Box />
      <Box />
      <Box />
    </Inline>
  </Story>
</Preview>

## switchAt

The below example will switch to a stack when it is less than `45rem`. (Resize your window to see this in action.)

<Preview withToolbar>
  <Story name="switchAt">
    <Inline gutter="lg" switchAt="45rem">
      <Box />
      <Box />
      <Box />
      <Box />
    </Inline>
  </Story>
</Preview>

## data-attribute

For purposes of styling, you can select `data-bedrock-layout-inline`.
