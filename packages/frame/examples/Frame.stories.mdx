# Frame

## Use case

The Frame is useful for cropping content, typically media, to a desired aspect ratio.

## How to install

If you haven't already, install `styled-components`:

```bash
  npm install styled-components
```

Then install the Frame component

```bash
  npm install @bedrock-layout/frame
```
