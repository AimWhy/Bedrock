# Stack

## Use case

The `Stack` is designed to literally 'stack' items on top of each other while maintaining a consistent gutter between each item.

## API

## Gutters

The `gutter` prop defines the spacing between elements. Bedrock has implemented a default spacing scheme, but [it can be overridden using the ThemeProvider provided by `styled-components`.](/docs/spacing--page#integrating-with-your-design-system)

The following values are available by default:
