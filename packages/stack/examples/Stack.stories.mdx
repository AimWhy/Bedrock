# Stack

## Use case

The Stack is designed to literally 'stack' items on top of each other while maintaining a consistent gutter between each item.

## Basic usage

<Preview withToolbar>
  <Story name="Basic">
    <Stack gutter="lg">
      <Box />
      <Box />
      <Box />
      <Box />
    </Stack>
  </Story>
</Preview>

## With xx-large gutters

<Preview withToolbar>
  <Story name="With xx-large gutters">
    <Stack gutter="xxl">
      <Box />
      <Box />
      <Box />
      <Box />
    </Stack>
  </Story>
</Preview>

## data-attribute

For purposes of styling, you can select `data-bedrock-layout-stack`.
