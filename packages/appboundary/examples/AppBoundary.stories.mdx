# AppBoundary

When we think of responsive design, we typically think of phones and tablets. However, screen resolutions are getting larger and allowing a page to grow to the same width as a 4K monitor is irresponsible. A website should clamp at an outer limit and stay centered on the screen.

## Use case

The `AppBoundary` component is designed to wrap your entire app. By default, it will clamp the width at the `xxlarge` size option, but can also be set to other the valid size options using the `boundarySize` prop. The `boundarySize` prop can also accept any positive integer or CSSLength as a string.

## Installation

```bash
yarn add @bedrock-layout/appboundary
```
