# Cover

## Use case

The Cover is designed to cover a predefined area, 100vh by default, and vertically center its child. You can also conditionally render a top and bottom section as well.

## API

## With a React Node as the child.

(All examples below are with the `minHeight` prop set to `50vh`)

The `Cover` component can be used with any React node as its child. The child will be vertically centered.

<Preview withToolbar>
  <Story name="With children">
    <Cover minHeight="50vh" gutter="sm">
      <div>
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Fuga

        voluptatibus tempore est dolor! Iusto, vero.
      </div>
    </Cover>
  </Story>
</Preview>

## data-attribute

For purposes of styling, the outer wrapper as the `data-bedrock-cover` data attribute. The `top`, `children`, and `bottom` are wrapped in an elements with a data attributes of `data-bedrock-cover-top`, `data-bedrock-cover-centered`, and `data-bedrock-cover-bottom` respectively.
