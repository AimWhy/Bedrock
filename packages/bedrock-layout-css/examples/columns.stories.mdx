# columns.css

This is a CSS Only version of the [Column package](/?path=/docs/components-columns--api).

## Use case

The Columns component is designed to create a n-column layout. The complimentary Column will allow elements to span and offset n-columns.

## Usage

First you need to import the appropriate stylesheet. The inline is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import '@bedrock-layout/bedrock-layout-css/lib/bedrock-layout.css';

// or

import "@bedrock-layout/bedrock-layout-css/lib/components/columns.css";
```

Then you add the `data-bedrock-columns` and `data-bedrock-column` attributes to the desired elements.

## --gutter

The `--gutter` prop defines the gutter size between elements. Bedrock has implemented a [default spacing scheme](css-only-components-spacing-properties-css--page), or you can simply pass any valid CSS length or percentage value.

## --columns

By default, the number of columns will be one. You can set the number of columns by setting the `--columns` property in the inline style.

The below example is a 4 column layout. Each box will take up one column by default.

</Preview>

## --span

By adding the `data-bedrock-column` attribute to an element, it will be styled as a column component.

You can adjust the number of columns that the element spans by setting the `--span` property in the inline style.

The column component is wrapping the blue and green boxes. It is then setting the blue box to span 3 columns and the green to span 2

<Preview withToolbar withSource="open">
  <Story name="--span">
    <div data-bedrock-columns style={{ "--gutter": "1rem", "--columns": 4 }}>
      <Box />
      <Box />
      <div data-bedrock-column style={{ "--span": 3 }}>
        <Box style={{ background: "blue" }} />
      </div>
      <div data-bedrock-column style={{ "--span": 2 }}>
        <Box style={{ background: "green" }} />
      </div>
      <Box />
      <Box />
      <Box />
    </div>

</Preview>

## offsetStart and offsetEnd

If you want the have the element offset by `n` number of columns, before or after , you can set the `--offsetStart` or `--offsetEnd` property in the inline style of the `data-bedrock-column` element.

In the below code, the blue box is `--offsetStart` of 1 and the green box has an `--offsetEnd` of 2

<Preview withToolbar withSource="open">

      <Box />
