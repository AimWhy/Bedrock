# center.css

This is a CSS Only version of the [Center package](/?path=/docs/components-center--api).

## Use case

The Center component is designed to center and clamp the width at a defined width, the medium breakpoint by default. You can also center the children and text alignment as well.

## Usage

First you need to import the appropriate stylesheet. The center is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import '@bedrock-layout/bedrock-layout-css/lib/bedrock-layout.css';

// or

import "@bedrock-layout/bedrock-layout-css/lib/components/center.css";
```

Then you add the `data-bedrock-center` attribute to the desired element.

## --maxWidth

The Center component will clamp the width at a defined `--maxWidth` and center itself in it's context. Please note that the `--maxWidth` is the max-width of the content and not the Center component itself.

You can adjust the max-width by setting the `--maxWidth` property in the inline style.

The below `--maxWidth` is set to `75%` of the parent component width.

## center-children

You can also set the children to center by adding `center-children` to the `data-bedrock-center` attribute. This will center the children within the element.

## center-text

You can also set the text to center by adding `center-text` to the `data-bedrock-center` attribute. This will center the text within the element.

        eu, elementum leo. Proin laoreet eleifend nisl ut iaculis. Ut dictum est

      </div>
    </div>
  </Story>
</Canvas>

Both properties can be combined in any order:

```html
<div data-bedrock-center="center-text">
  <!-- text alignment will be centered -->
</div>

<div data-bedrock-center="center-children">
  <!-- children will be centered -->
</div>

<div data-bedrock-center="center-text center-children">
  <!-- text alignment and children will be centered -->
</div>
```
