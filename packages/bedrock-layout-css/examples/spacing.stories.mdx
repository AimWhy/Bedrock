# Spaceing Properties

## EXPERIMENTAL

This is a list of CSS custom properties used in the [Bedrock Layout](?path=/docs/spacing--page) project.

```javascript
import "@bedrock-layout/css-reset/lib/components/spacing-properties.css";
```

Here is a list of the current properties and their values:

```css
:root {
  --space-none: 0px;
  --space-xxs: 0.0625rem;
  --space-xs: 0.125rem;
  --space-sm: 0.25rem;
  --space-md: 0.5rem;
  --space-mdLg: 0.75rem;
  --space-lg: 1rem;
  --space-lgXl: 1.5rem;
  --space-xl: 2rem;
  --space-xlXXl: 3rem;
  --space-xxl: 4rem;

  --size-xxsmall: 159px;
  --size-xsmall: 319px;
  --size-small: 639px;
  --size-medium: 1023px;
  --size-large: 1199px;
  --size-xlarge: 1439px;
  --size-xxlarge: 1920px;
}
```
