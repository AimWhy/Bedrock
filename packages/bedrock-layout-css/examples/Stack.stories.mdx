# CSS Reset

## EXPERIMENTAL

This is a pure-css version of the [Stack package](/?path=/docs/components-stack--api).

```javascript
import "@bedrock-layout/css-reset/lib/components/stack.css";
```

## Usage

By adding the `data-bedrock-stack` attribute to an element, it will be styled as a stack.

You can adjust the space between the child elements by setting the `--gutter` property in the inline style.

```html
<div data-bedrock-stack style="--gutter:2rem">
  <!-- content goes here -->
</div>

<div data-bedrock-stack style="--gutter:var(--space-mdLg)">
  <!-- content goes here -->
</div>
```
