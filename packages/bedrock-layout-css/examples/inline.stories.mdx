# inline.css

This is a CSS Only version of the [Inline package](/?path=/docs/components-inline--api).

## Use case

The Inline component is designed to create consistent spacing around inline elements of variable width. Unlike the div component,
the items in the Inline component will not wrap be default. The Inline component also allows you to specify a child that can stretch, filling the excess space.

## Usage

First you need to import the appropriate stylesheet. The inline is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import '@bedrock-layout/bedrock-layout-css/lib/bedrock-layout.css';

// or

import "@bedrock-layout/bedrock-layout-css/lib/components/inline.css";
```

Then you add the `data-bedrock-inline` attribute to the desired element.

## --gutter

The `--gutter` prop defines the gutter size between elements. Bedrock has implemented a [default spacing scheme](css-only-components-spacing-properties-css--page), or you can simply pass any valid CSS length or percentage value.

## justify

The `justify` prop defines the inline justification of the elements within the cluster. It accepts the following values: `justify:start`, `justify:end`, `justify:center`.

You set the justification by setting the `justify` property on the `data-bedrock-inline` attribute.

Both the `justify` and `align` properties can be set to at the same time.

```html
<div
  data-bedrock-inline="justify:start align:center"
  style="--gutter:var(--space-mdLg)"
>
  <!-- content goes here -->
</div>
```

## stretch

The `stretch` prop can be used to set one or all of the children to stretch and fill the available space in the inline direction. You can set the `stretch` property on the `data-bedrock-inline` attribute.

The `stretch` prop accepts the following values: `stretch:start`, `stretch:end`, `stretch:all`.

Unlike the `Inline` component, the `stretch` property can't be set to any integer value. The `stretch` property can only be set to 0-4 to represent the first, second, third, fourth, or fifth child respectively.

## --switchAt

If you set the `--switchAt` property, the inline will switch to a stacking layout if the width of the element is below the threshold set by the `--switchAt` property.

The below example will switch to a stack when it is less than `45rem`.

(Resize your window to see this in action.)
