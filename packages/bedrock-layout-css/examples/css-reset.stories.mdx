# CSS Reset

This is a CSS Only version of the [CSS reset package](/?path=/docs/css-reset--page).

A CSS Reset that compliments well with the Bedrock Layout Primitives

The reset is already included in the main CSS bundle, or you can bring it in individually.

```javascript
import '@bedrock-layout/bedrock-layout-css/lib/bedrock-layout.css';

// or

import "@bedrock-layout/bedrock-layout-css/lib/components/reset.css";
```
