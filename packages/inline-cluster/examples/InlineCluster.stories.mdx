# InlineCluster

## Use case

The InlineCluster component is used to display a group of elements in a row. When the group is too large to fit on a single line, the elements are displayed in a cluster based on the width of the container and the justification of the cluster.

## gutter

The `gutter` prop defines the gutter size between elements. Bedrock has implemented a default spacing scheme, but [it can be overridden using the ThemeProvider provided by `styled-components`.](/docs/spacing--page#integrating-with-your-design-system)

## justify

The `justify` prop defines the inline justification of the elements within the cluster. It accepts the following values: `start`, `end`, `center`, `stretch`.

## align

The `align` prop defines the block alignment of the elements within the cluster. It accepts the following values: `start`, `end`, `center`, `stretch`.

For purposes of styling, you can select `data-bedrock-inline-cluster`.
