# MasonryGrid

## Use case

The `MasonryGrid` component is almost identical to the [Grid component](/?path=/docs/components-grid--api) except that each item's vertical space will grow independant of each other. The `MasonryGrid` component will then optimize the number of columns based on the `minItemWidth` prop value passed in. `MasonryGrid` does not create standard rows. Instead, it will optimize for the most dense layout that it can achieve based on the space available.

## Installation

```bash
yarn add @bedrock-layout/masonry-grid
```
