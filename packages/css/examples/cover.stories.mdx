# cover.css

This is a CSS Only version of the [Cover package](/?path=/docs/components-cover--api).

## Usage

First you need to import the appropriate stylesheet. The cover is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import '@bedrock-layout/css/lib/bedrock-layout.min.css';

// or

import "@bedrock-layout/css/lib/components/cover.min.css";
```

Then you add the `data-bedrock-cover` attribute to the desired element.

## data-bedrock-cover-centered

The cover lets you center an element vertically in a container.
To achieve that, you need to add the `data-bedrock-cover-centered` attribute to the element that needs to be vertically centered.
You can add content above the centered element, below it, or both. In that case the element will be centered between the content above and below.

By default, the minimum height of the `data-bedrock-cover` element is `100vh`. You can update this by setting the `--minHeight` property in the inline style.

(The below examples with the `--minHeight` prop set to `25vh`)

## --gutter

The `--gutter` prop defines the gutter size between elements. Bedrock has implemented a [default spacing scheme](css-only-components-spacing-properties-css--page), or you can simply pass any valid CSS length or percentage value.

This will ensure space between the elements if the content is larger than the `--minHeight` prop.

(The below example, the `--minHeight` prop is set to `10vh` and the `--gutter` prop is set to `2rem`)

<Canvas withSource="open">
  <Story name="--gutter">
    <div
      data-bedrock-cover
      style={{
        "--gutter": "2rem",
        "--minHeight": "10vh",

## --minHeight

The `--minHeight` custom property can be used to set the minimum height of the cover. The default is `100vh`.

(The below example, the `--minHeight` prop is set to `500px` and the `--gutter` prop is set to `2rem`)

<Canvas withSource="open">
  <Story name="--minHeight">
    <div
      data-bedrock-cover
      style={{
        "--gutter": "2rem",
        "--minHeight": "500px",
        border: "1px solid black",
      }}
    >
      <span>I am on top.</span>
      <div data-bedrock-cover-centered>
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Fuga
        consequuntur corrupti beatae commodi vitae, perspiciatis totam provident

## stretch-content

Setting the `stretch-content` prop on the `data-bedrock-cover` attribute will stretch the centered content to fill the remaining height of the cover.

(The below example, the `--minHeight` prop is set to `50vh` and the `--gutter` prop is set to `2rem`)

<Canvas withSource="open">
  <Story name="stretch-content">
    <div
      data-bedrock-cover="stretch-content"
      style={{
        "--gutter": "2rem",
        "--minHeight": "500px",
        border: "1px solid black",
      }}
    >
      <span>I am on top.</span>
      <div
        data-bedrock-cover-centered
        style={{ background: "rgba(0,0,0,0.85)", color: "white" }}
      >
        Lorem ipsum dolor sit amet consectetur adipisicing elit. Fuga

        architecto doloribus aperiam sapiente, incidunt nihil suscipit

      <span>I am on bottom.</span>
    </div>
