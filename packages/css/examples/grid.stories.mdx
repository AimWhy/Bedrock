# grid.css

This is a CSS Only version of the [Grid package](/?path=/docs/components-grid--api).

## Use case

The Grid component is designed to create a responsive grid of items that will automatically wrap based on the quantity of children and the `--minItemWidth` value.

## Usage

First you need to import the appropriate stylesheet. The grid is part of the main CSS bundle or you can import it in individually from the components directory.

```javascript
import '@bedrock-layout/css/lib/bedrock-layout.min.css';

// or

import "@bedrock-layout/css/lib/components/grid.min.css";
```

Then you add the `data-bedrock-grid` attribute to the desired element.

## --gutter

The `--gutter` prop defines the gutter size between elements. Bedrock has implemented a [default spacing scheme](css-only-components-spacing-properties-css--page), or you can simply pass any valid CSS length or percentage value.

You can adjust the space between the child elements by setting the `--gutter` property in the inline style.

## --minItemWidth

The grid will automatically optimize the number of columns based on the width of the container and the value provided to the `--minItemWidth` property. By default this is set to `639px`, but you can change this value by setting the `--minItemWidth` property in the style.

In the below example, The `--minItemWidth` is set to `15rem`. As you resize the window, the Grid will recalculate and potentially change the count of columns and rows.
